Double 3D Checkers Game

This is a very simple double checkers game, which is played by 2 players. Eat all enemy's pieces to win the game. Include full source code so you can easily make an individual mode (as known as player vs. computer).

Features:
- Easy to win
- Ready to use 3D checkers pieces
- Include all basic piece movements (walk, eat, promotion)
- 'As much as possible' and 'one per moves' piece eating rule setting
- Open source
- Clean script interface

This asset does not include package manager dependencies, you need to import 'Unity UI' in package manager manually.
